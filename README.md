## 开源协议

由于不同脚本创建的时间和使用的环境不同，所以可能不是所有脚本都使用相同的开源协议(部分脚本也使用WTFPL等协议)。

如果脚本包含`@license`行，则认为对应脚本使用其指定的开源协议。

否则，默认使用GPLv3协议。
